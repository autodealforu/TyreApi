## 🧪 API Testing Endpoints

### Authentication

```bash
# Customer Login
POST /api/users/login
{
  "username": "customer@test.com",
  "password": "password123"
}

# Vendor Login
POST /api/users/login
{
  "username": "vendor@test.com",
  "password": "vendor123"
}
```

### Job Cards

```bash
GET /api/jobcards - Get all job cards
GET /api/jobcards/:id - Get specific job card
POST /api/jobcards - Create new job card
PUT /api/jobcards/:id - Update job card
DELETE /api/jobcards/:id - Delete job card
```

### Tyres & Products

```bash
GET /api/tyres - Get all tyres
GET /api/products - Get all products
GET /api/tyres/:id - Get specific tyre
GET /api/products/:id - Get specific product
```
